# 📖 PromptHub 使用指南

## 什么是 PromptHub？

PromptHub 是一个免费的 AI 提示词模板市场，收录了来自 ChatGPT、Claude、Midjourney 等主流 AI 平台的优质提示词。用户可以快速找到需要的提示词，一键复制即可使用。

## ❓ 常见问题

### Q: 这些提示词免费吗？
A: 是的，PromptHub 上所有提示词模板都是免费的。

### Q: 可以商用吗？
A: 仅供个人学习参考使用，商业使用请注意各 AI 平台的使用条款。

### Q: 如何提交新的提示词模板？
A: 欢迎提交！请参考 GitHub 仓库中的贡献指南。

### Q: 提示词中的 {} 是什么意思？
A: {} 中的内容是需要你替换的变量，根据实际情况填入即可。

## 🛠️ 本地开发

```bash
# 克隆项目
git clone https://github.com/Answer-version/prompt-market.git

# 进入目录
cd prompt-market

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
