## ✨ 特性

- 📚 **海量模板** - 收录 30+ 优质提示词，覆盖主流 AI 平台
- 📋 **一键复制** - 点击即可复制提示词，立刻使用
- 🆓 **完全免费** - 所有模板免费使用，无需注册
- 📱 **响应式设计** - 手机、电脑都能流畅访问
- ❤️ **收藏夹** - 收藏喜欢的模板，随时查阅
- 🔄 **持续更新** - 不断添加新的优质模板

## quick start

```bash
# 克隆项目
git clone https://github.com/Answer-version/prompt-market.git
cd prompt-market

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 🤝 如何贡献

欢迎提交模板！

1. Fork 本项目
2. 创建新分支 (`git checkout -b feature/new-prompt`)
3. 在 `src/data/prompts.js` 中添加新模板
4. 提交更改 (`git commit -m 'feat: 添加 xxx 模板'`)
5. Push 到分支 (`git push origin feature/new-prompt`)
6. 创建 Pull Request
