# 💡 PromptHub - AI提示词模板市场

> 发现和分享高质量AI提示词模板 | 涵盖ChatGPT、Claude、Midjourney、Stable Diffusion等平台

## ✨ 特性

- 📚 **海量模板** - 收录各类优质提示词，覆盖职场、编程、创作等多个场景
- 🔍 **智能筛选** - 按平台、分类快速找到需要的模板
- 📋 **一键复制** - 点击即可复制提示词，立即使用
- 🆓 **免费优先** - 大部分模板免费使用
- 📱 **响应式设计** - 手机电脑都能用

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/Answer-version/prompt-market.git
cd prompt-market

# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```
